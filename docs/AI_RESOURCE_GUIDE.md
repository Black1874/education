# 儿童教育平台资源清单（AI生成版）

## 使用说明

1. 将下面的AI提示词复制到DALL-E、Midjourney或Stable Diffusion
2. 生成图片后，按照"文件名"保存到对应目录
3. 所有图片统一规格：512x512px，PNG格式，透明背景
4. 替换后刷新页面即可看到效果

## 批量生成脚本

### 使用方法

1. 复制下面的清单到Excel或CSV
2. 使用Python脚本批量调用AI API
3. 或者手动复制提示词到AI工具

### 文件清单（CSV格式）

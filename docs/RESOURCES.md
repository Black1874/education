## 资源规格要求

### 图片规格
- **格式**: PNG（透明背景）
- **尺寸**: 512x512px（学习内容）、256x256px（图标）
- **风格**: 卡通、可爱、色彩鲜艳但柔和
- **文件大小**: 单张 < 200KB

### 音频规格
- **格式**: MP3
- **比特率**: 128kbps
- **时长**: 
  - 动物叫声: 1-3秒
  - 名称语音: 1-2秒
  - 鼓励音效: 1-2秒
  - 背景音乐: 30-60秒（循环）
- **文件大小**: 单个 < 100KB

## 资源来源建议

### 免费资源网站
1. **图片**:
   - Freepik (https://www.freepik.com/) - 大量儿童插画
   - Flaticon (https://www.flaticon.com/) - 图标资源
   - unDraw (https://undraw.co/) - 可自定义颜色的插画
   - Pixabay (https://pixabay.com/) - 免费图片

2. **音效**:
   - Freesound (https://freesound.org/) - 免费音效
   - Zapsplat (https://www.zapsplat.com/) - 音效库
   - Mixkit (https://mixkit.co/free-sound-effects/) - 免费音效

3. **AI生成**:
   - DALL-E / Midjourney - AI生成图片
   - ElevenLabs - AI生成语音
